# tarkov-simulator

Escape from Tarkov 시즌 1 KORD BREACH의 특성 포인트 시뮬레이터. 게임에서는 모디파이어라고 부른다. Project Zomboid처럼 긍정 특성은 포인트를 소모하고 부정 특성은 포인트를 획득하는 방식이라, 시즌 시작 전에 원하는 빌드를 미리 짜 볼 수 있다.

## 기능

- 글로벌 모디파이어 6종은 참고용으로 표시하고, 긍정 17종과 부정 13종은 카드 클릭으로 선택 토글
- 긍정 소모, 부정 획득, 순 합계 포인트 실시간 계산
- 같은 스탯을 정반대로 미는 특성 7쌍은 동시 선택 차단. 이 충돌 목록은 공식 정보가 아닌 추정이다
- 선택 전체 초기화

## 동작 방식

단일 페이지 클라이언트 컴포넌트로 동작한다. 특성 데이터는 `src/data/traits.ts`에 하드코딩되어 있고 외부 API나 DB를 쓰지 않는다. 데이터 출처는 `docs/KORDBREACH.md`이며, 수치는 시즌 시작 전에 바뀔 수 있다.

## 실행

의존성 설치 후 개발 서버를 띄운다. 환경변수는 필요 없다.

```bash
pnpm install
pnpm dev
```

`http://localhost:3000`에서 확인한다. 프로덕션 빌드는 다음과 같다.

```bash
pnpm build
pnpm start
```
